\# 🤖 HCML Telegram Bot



\[!\[Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://python.org)

\[!\[Telegram](https://img.shields.io/badge/Telegram-Bot-blue.svg)](https://t.me/HCML\_Bot)



ربات تلگرام برای \*\*Hanzi Cipher Markup Language (HCML)\*\* - رمزنگاری خلاقانه متن با استفاده از کاراکترهای چینی!



\## ✨ قابلیت‌ها



\- 🔐 رمزنگاری متن به کاراکترهای چینی

\- 🔓 رمزگشایی متن‌های رمز شده

\- 📚 پشتیبانی از 97,600 کاراکتر چینی

\- ⚙️ تنظیمات پیشرفته (key, count, way)

\- 💾 ذخیره‌سازی کلاس‌های سفارشی

\- 🎨 رابط کاربری زیبا با دکمه‌های تعاملی
